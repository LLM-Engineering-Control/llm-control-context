# Domain Model

## 한국어

초기 도메인 엔티티는 아래를 기준으로 합니다.

- `User`
- `Workspace`
- `Service`
- `Incident`
- `IncidentComment`
- `IncidentActionLog`
- `NotificationLog`

### 상태 예시

`Incident.status`

- `OPEN`
- `INVESTIGATING`
- `MITIGATED`
- `RESOLVED`

### 우선순위 예시

- `LOW`
- `MEDIUM`
- `HIGH`
- `CRITICAL`
