# Baseline Setup

## 한국어

이 문서는 실험 시작 전에 완료해야 하는 공통 준비 단계를 정의합니다.
이 단계의 작업은 비교 측정 대상에 포함하지 않습니다.

### 비교 제외 범위

- 저장소 생성
- 브랜치 전략 정리
- GitHub 이슈/PR 템플릿 생성
- 모노레포 폴더 구조 생성
- Spring Boot 기본 스캐폴딩
- React + TypeScript 기본 스캐폴딩
- 공통 문서 초안 작성

### 실험 시작 시점

- `001-auth-login` 이슈부터 측정 시작

### 시작 조건

- `main`, `dev`, `feat/*` 전략 준비 완료
- README 정리 완료
- 백엔드/프론트 기본 실행 구조 준비 완료
- 메트릭 기록 파일 준비 완료
- 컨텍스트 문서 초안 준비 완료

## English

This document defines the common setup tasks that must be completed before the experiment starts.
Work in this phase is not included in measured comparison results.

### Excluded From Measurement

- repository creation
- branch strategy setup
- GitHub issue and PR templates
- monorepo folder creation
- Spring Boot base scaffolding
- React and TypeScript base scaffolding
- initial shared documentation

### Experiment Start Point

- measurement starts from issue `001-auth-login`

### Start Conditions

- `main`, `dev`, and `feat/*` strategy is ready
- README is prepared
- backend and frontend base app structure exists
- metrics file exists
- context document draft exists
